# Summary

  * debug/trace
  * colored cat equivalents
  * object properties, e.g. `has_name`, `is_empty`
  * indexing, e.g. `first`, `last`
  * STL-like containers
  * purrr-like `map*` functions
  * other

# Installation

```r
install_github("lbartnik/utilities")
```
